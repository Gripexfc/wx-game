# 慢慢变乖鸭排版与品牌重构 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把首页、首次引导页、任务页整体重排成更适合手机用户的小程序体验，统一 `慢慢变乖鸭` 品牌与“小鸭 + 用户命名”的角色关系，并移除这三个核心页面中的 Banner 广告展示。

**Architecture:** 保持现有 Canvas UI 渲染方式和页面类边界不变，优先在现有页面文件内重排布局、重写文案与弱化高密度信息区。为了让改动可测，先把品牌文案和页面布局参数提取成可验证的纯函数/配置，再按 TDD 更新页面渲染与交互，最后回归人工检查首屏观感、滚动节奏与广告移除效果。

**Tech Stack:** 微信小游戏 Canvas UI、CommonJS 模块、Node 直跑测试脚本、现有 `BannerAdManager`

```javascript
// js/ui/pageLayoutSpec.js
function getBrandCopy(duckName) {
  const safeName = duckName || '你的小鸭';
  return {
    brandTitle: '慢慢变乖鸭',
    relationshipLine: `今天也和 ${safeName} 一起慢慢变好`,
  };
}

function getHomePageLayoutSpec(width, height) {
  const horizontalPadding = 20;
  const petCardHeight = Math.floor(height * 0.40);
  const actionAreaHeight = Math.floor(height * 0.26);
  return {
    horizontalPadding,
    petCardHeight,
    actionAreaHeight,
    showBottomBanner: false,
  };
}

function getOnboardingCopy() {
  return {
    title: '慢慢变乖鸭',
    subtitle: '领一只会陪你慢慢变好的小鸭',
    inputPlaceholder: '给它起个名字',
    confirmText: '带它回家',
  };
}

function getTaskPageCopy(duckName) {
  const safeName = duckName || '你的小鸭';
  return {
    title: `${safeName}今天的小计划`,
    summary: `和 ${safeName} 一起完成`,
  };
}

module.exports = {
  getBrandCopy,
  getHomePageLayoutSpec,
  getOnboardingCopy,
  getTaskPageCopy,
};
```
