# 《噜噜陪你成长》微信小游戏设计文档

## 1. 产品定位

**游戏名称：** 噜噜陪你成长
**类型：** 治愈养成·每日签到
**核心用户：** 18-40 岁希望养成好习惯的年轻人
**产品口号：** "每一天努力，都有人陪伴"
**核心价值：** 用可爱水豚噜噜陪伴你养成好习惯，完成日常任务获得成长与治愈

---

## 2. 视觉设计规范

### 2.1 配色方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色（水豚棕） | 温暖棕 | #8B7355 |
| 辅助色（草地绿） | 清新绿 | #7DCEA0 |
| 点缀色（温暖黄） | 阳光黄 | #F5B041 |

| 文字主色 | 深棕 | #5D4E37 |
| 文字辅色 | 浅棕 | #8B7355 |

### 2.2 角色设计：噜噜（水豚）

**基础形象：**
- 圆滚滚的水豚身体（椭圆形）
- 小眼睛、大鼻子、耳朵
- 四条小短腿
- 原色为温暖棕色 #8B7355

**4 成长阶段：**

| 阶段 | 名称 | 外观特征 | 解锁等级 |
|------|------|----------|----------|
| Stage 1 | 幼崽噜噜 | 小巧圆润，眼睛占比大 | Lv 1-5 |
| Stage 2 | 少年噜噜 | 稍大，眼神更坚定 | Lv 6-10 |
| Stage 3 | 青年噜噜 | 体型健壮，开始有配饰槽 | Lv 11-15 |
| Stage 4 | 成体噜噜 | 完整水豚形态，可带多配饰 | Lv 16-20 |

### 2.3 UI 设计风格

- **整体基调：** 温暖治愈、手绘感、圆润可爱
- **圆角设计：** 所有卡片/按钮统一圆角 16-20px
- **微动效：** 噜噜呼吸动画、点击反馈、任务完成庆祝
- **单手操作：** 核心操作在屏幕下半部分

## 3. 核心玩法

### 3.1 首页设计

**首页即是噜噜展示页：**
- 中央展示噜噜当前形态（大尺寸，占据 50% 屏幕）
- 噜噜有idle动画（轻微摇晃/呼吸）
- 点击噜噜触发互动（噜噜发出声音/做动作）
- 右上角：今日任务入口按钮
- 左上角：成就/图鉴入口
- 底部：经验条、等级显示

### 3.2 每日任务系统

**任务类型（固定4种 + 自选）：**

| 任务 | 图标 | 描述 | 奖励经验 |
|------|------|------|----------|
| 健身 | 🏃 | 完成30分钟运动 | +20 XP |
| 吃饭 | 🍽️ | 记录正餐 | +15 XP |
| 阅读 | 📚 | 阅读30分钟 | +20 XP |
| 睡眠 | 😴 | 早睡打卡(22:00前) | +25 XP |

**每日任务规则：**
- 每日0点刷新任务
- 固定任务：每天4个自动发布
- 自选任务：用户可选2-3个额外任务
- 完成任务获得经验值
- 任务可补签（限当天24:00前）

### 3.3 成长系统

**经验值（XP）获取：**
- 完成任务：+15~25 XP
- 连续签到：每天+5 XP bonus
- 完美一天（全部完成）：+30 XP bonus

**升级所需 XP：**
- Lv 1→2: 100 XP
- Lv 2→3: 150 XP
- Lv 3→4: 200 XP
- ... 逐级递增

**升级奖励：**
- 每升1级：获得 1 颗爱心（用于解锁配饰）
- 每升5级：解锁新配饰槽或场景

### 3.4 噜噜互动系统

**点击噜噜触发：**
- 随机互动动作（5种基础动作）
- 噜噜会根据今日任务完成情况说话
- 完成任务后：噜噜会说"太棒了！"、"你好厉害~"等夸奖
- 未完成：噜噜会说"加油哦~"、"我陪着你~"等鼓励

**互动动作：**
- 摇头晃脑
- 眯眼微笑
- 蹭蹭撒娇
- 蹦蹦跳
- 打哈欠

## 4. 收集与奖励系统

### 4.1 配饰系统（外观收集）

**配饰类型：**

| 类型 | 示例 | 解锁方式 |
|------|------|----------|
| 帽子 | 棒球帽、草帽、圣诞帽 | 升级奖励 |
| 围巾 | 格子围巾、毛绒围巾 | 成就奖励 |
| 眼镜 | 圆眼镜、墨镜 | 连续签到 |
| 背包 | 小书包、竹筐 | 完成任务累计 |
| 食物 | 胡萝卜、苹果 | 随机掉落 |

**配饰槽位：**
- 帽子槽（1个）
- 围巾槽（1个）
- 眼镜槽（1个）
- 背包槽（1个）

### 4.2 场景系统

### 4.3 成就系统

| 满级主人 | 达到Lv 20 | 金色噜噜框 |

## 5. 界面结构

### 5.1 界面列表

1. **首页（噜噜展示）** - 噜噜主视觉、互动、状态栏
2. **今日任务页** - 固定任务 + 自选任务列表
3. **噜噜图鉴页** - 成长阶段展示、配饰收集、场景解锁
4. **成就页** - 已解锁/未解锁成就墙
5. **设置页** - 提醒时间、音效、关于

### 5.2 导航流程

```
首页（噜噜）
   ├→ 点击噜噜 → 互动动画/说话
   ├→ 右上角按钮 → 今日任务页
   │                 ├→ 完成任务 → 奖励 XP
   │                 └→ 自选任务 → 添加任务
   ├→ 左上角按钮 → 图鉴页
   │                 ├→ 配饰穿戴
   │                 └→ 场景选择
   └→ 成就页
```

## 6. 技术方案

### 6.1 技术栈

- **渲染：** 微信原生 Canvas API
- **语言：** JavaScript（ES6+）
- **适配：** 微信小游戏适配层
- **存储：** wx.getStorageSync 本地存储

### 6.2 项目结构

```
/wx-game
├── game.js              # 游戏主入口
├── js/
│   ├── main.js          # 游戏主类
│   ├── Lulu.js         # 噜噜角色类
│   ├── Task.js         # 任务系统
│   ├── Growth.js        # 成长/XP系统

├── utils/
│   ├── constants.js   # 常量
```
